# 🌐 Innovate Digital Solutions (IDS)
เว็บไซต์สำหรับบริษัทผู้ให้บริการโซลูชั่นดิจิทัลและเทคโนโลยี ประกอบด้วยหน้าหลัก, หน้าเกี่ยวกับบริษัท, หน้าบริการ และหน้าติดต่อ

# ✨ ฟีเจอร์หลักของเว็บ (Key Features)
แสดงข้อมูลบริษัท Innovate Digital Solutions (IDS)

นำเสนอผู้บริหาร (กรรมการบริษัทคือ ลัดดา สิทธิกะษณะ)

แสดงเป้าหมาย (Mission) และวิสัยทัศน์ (Vision)

แสดงบริการหลัก 3 ด้าน (บริการดิจิทัลแพลตฟอร์ม, กลยุทธ์การตลาด, และการวางแผนและจัดการการตลาด)

มีตารางเปรียบเทียบบริการ

มีแบบฟอร์มติดต่อและช่องทางติดต่อเพิ่มเติม (เช่น โทรศัพท์, อีเมล, Instagram, Facebook)

มีเมนูนำทางเชื่อมทุกหน้า
